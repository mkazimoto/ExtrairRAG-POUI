# po-dynamic

### Enum `ForceBooleanComponentEnum`

/**
@usedBy PoDynamicFormComponent

@description

Enum para definição do tipo de componente a ser renderizado.
/

- `switch = 'switch'` — Força a renderização de um po-switch
- `checkbox = 'checkbox'` — Força a renderização de um po-checkbox

---

### Enum `ForceOptionComponentEnum`

/**
@usedBy PoDynamicFormComponent

@description

Enum para definição do tipo de componente a ser renderizado.
/

- `radioGroup = 'radioGroup'` — Força a renderização de um po-radio-group independente da quantidade do opções
- `select = 'select'` — Força a renderização de um po-select independente da quantidade do opções

### Enum `PoDynamicFieldType`

/**
@usedBy PoDynamicFormComponent

@description

Enum para definição do tipo de campo que será criado dinamicamente.
/

- `Boolean = 'boolean'` — Valor booleano.
- `Currency = 'currency'` — Valor numérico que contém casas decimais e milhar.
- `Decimal = 'decimal'` — Valor numérico que contém casas decimais e milhar.
- `Date = 'date'` — Valor para data.
- `DateTime = 'datetime'` — Valor para data e hora.
- `Time = 'time'` — Utilizado para informar/exibir hora.
- `Number = 'number'` — Valor numérico.
- `String = 'string'` — Texto.
- `Upload = 'upload'` — Utilizado para fazer uploads de arquivos.
